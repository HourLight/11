複方按摩調理油（教學基礎配方）

教學用油｜共3瓶｜500ml/瓶

基礎配方成分

沙棘果油（基底油） 基底
橄欖油（基底油） 基底
複方精油（總濃度3%） 3%

📝 複方精油組成：冬青、真正薰衣草、薄荷、乳香、胡荽葉、馬郁蘭、羅馬洋甘菊、羅勒、迷迭香
⭐ 教學重點：學員可依據不同需求，在此基礎配方上增添單方精油進行客製化調理

基礎使用方法

  1. 使用前請先輕輕搖晃瓶身，使油脂均勻混合
  2. 按壓1-2次於手心（約3-5ml），雙手掌心互相摩擦溫熱
  3. 將溫熱的調理油輕柔塗抹於需要照護的身體部位
  4. 以緩慢畫圓的方式輕柔按摩，幫助油脂吸收與氣血循環
  5. 建議在沐浴後使用，此時毛孔較為舒張，吸收效果較佳
  6. 使用後可稍作休息，讓身體充分吸收植物精華

💡 進階教學應用：學員在熟悉基礎配方後，可依據個人狀況或服務對象需求，適度添加其他單方精油（如：天竺葵、絲柏、檸檬草等），調整出更符合個別化需求的配方

重要注意事項

  • 敏感測試必做：首次使用前，請先取少量塗抹於手腕內側或耳後，觀察15-30分鐘，確認無紅腫、發癢等不適反應再使用
  • 特殊族群請避用：懷孕期間、哺乳中、嬰幼兒（3歲以下）、蠶豆症患者請避免使用
  • 避開敏感部位：請勿接觸眼睛、口腔、鼻腔等黏膜部位，若不慎接觸請立即以大量清水沖洗
  • 本配方含薄荷成分：對薄荷過敏者請勿使用；使用後避免立即接觸眼睛周圍
  • 本配方含光敏性成分：使用後4-6小時內避免陽光直接曝晒
  • 儲存方式：請放置於陰涼乾燥處，避免陽光直射與高溫環境，並遠離兒童可觸及之處
  • 保存期限：開封後建議於6個月內使用完畢，以確保植物精油品質穩定
  • 若有不適請停用：使用過程中若出現任何不適症狀（如：紅腫、刺痛、頭暈等），請立即停止使用並以清水沖洗，必要時尋求專業協助

臉部卸妝按摩調理油

教學用油｜共3瓶｜100ml/瓶

基礎配方成分

葡萄籽油（基底油） 基底
沙棘果油（基底油） 基底
Tween20（天然乳化劑） 乳化劑
複方精油（總濃度3%） 3%

📝 複方精油組成：真正薰衣草、檸檬、茶樹
🔬 Tween20說明：天然來源的乳化劑，幫助油脂與水分結合，使卸妝油能順利乳化沖洗乾淨

正確使用步驟（非常重要！）

  1. 取油：使用滴管取適量（約5-8滴，依臉部面積調整）於乾燥的手心
  2. 塗抹：雙手搓勻後，輕柔塗抹於乾燥的臉部肌膚（包含眼周彩妝，但避開眼睛內部）
  3. 按摩溶解：以指腹輕柔畫圓按摩約8-10分鐘，讓油脂充分溶解彩妝與深層污垢
  4. 加水乳化（關鍵步驟）：手沾取少量溫水，在臉上繼續按摩，此時油脂會開始變成乳白色
  5. 完全乳化：持續加水按摩，直到整臉的油完全變成乳白色霜狀（這個步驟非常重要！）
  6. 清水沖洗：確認完全乳化後，用大量溫水沖洗乾淨
  7. 二次清潔：建議使用溫和洗面乳再次清潔，確保無殘留

⚠️ 乳化不完全的後果：
若沒有完全乳化就沖水，油脂會殘留在臉上，可能導致毛孔阻塞或產生悶痘現象。請務必確認整臉都呈現乳白色霜狀後再沖洗！

💡適用情境

  • ✓ 濃妝卸除（粉底液、遮瑕、眼影、眼線等）
  • ✓ 防曬產品卸除（特別是防水型防曬）
  • ✓ 臉部深層污垢清潔（毛孔粉刺、油脂堆積）
  • ✓ 結合臉部按摩手技，促進循環調理（按摩時間可延長至10-15分鐘）

重要注意事項

  • 敏感測試必做：首次使用前，請先取少量塗抹於手腕內側，觀察15-30分鐘，確認無不適反應再用於臉部
  • 避開眼睛內部：雖可用於眼周卸妝，但請避免讓油脂進入眼睛內部，若不慎入眼請立即以清水沖洗
  • 必須完全乳化：這是最關鍵的步驟！未完全乳化會導致油脂殘留，可能造成毛孔阻塞
  • 特殊族群請避用：懷孕期間、哺乳中、嬰幼兒（3歲以下）請避免使用
  • 本配方含檸檬精油（光敏性成分）：使用後12小時內避免陽光直接曝晒，建議於晚間使用
  • 油性肌膚與痘痘肌注意：使用頻率建議為每週2-3次，避免過度使用造成油脂負擔
  • 乾性肌膚友善：本配方較適合乾性至中性肌膚，使用後肌膚會較為滋潤
  • 二次清潔不可省：使用卸妝油後務必再用洗面乳清潔，確保無殘留
  • 隱形眼鏡使用者：建議先卸除隱形眼鏡後再使用本產品
  • 儲存方式：請放置於陰涼乾燥處，避免陽光直射與高溫，並遠離兒童可觸及之處
  • 保存期限：開封後建議於4-6個月內使用完畢（含光敏性成分，開封後較易氧化）
  • 若有不適請停用：使用過程中若出現紅腫、刺痛、發癢等不適症狀，請立即停止使用並以清水沖洗

頭皮養護調理油（教學基礎配方）

教學用油｜200ml×3瓶 + 50ml×3瓶

基礎配方成分

透氣油（基底油） 基底
小白菊花油（浸泡油） 基底
複方精油（總濃度3%） 3%

📝 複方精油組成：雪松、迷迭香、真正薰衣草、茶樹、乳香、羅馬洋甘菊、柑橘、甜橙
🌼 小白菊花油說明：又稱「母菊花浸泡油」，為天然植物浸泡油，有助於頭皮舒適調理

正確使用步驟（請務必依序完成）

  1. 乾髮使用：請在乾燥的頭髮和頭皮上使用（不要先沖濕）
  2. 分區塗抹：將頭髮分區，用指尖或滴管取適量調理油，直接塗抹於頭皮上（不是頭髮！）
  3. 均勻分布：用指腹輕柔地將油脂均勻分布於整個頭皮，特別是容易出油或有不適感的區域
  4. 按摩吸收：以指腹輕柔按摩頭皮3-5分鐘，促進吸收與循環
  5. ⏰ 關鍵步驟——靜置等待：按摩完成後，請靜置15-20分鐘，讓植物精油充分滲透並分解毛囊內的多餘油脂與污垢
  6. 可使用蒸氣或熱毛巾：靜置期間可用溫熱毛巾包覆或使用蒸氣帽，加強滲透效果（非必要步驟）
  7. 🚿 第一次清潔：時間到後，先用溫水沖濕頭髮，搭配洗髮精清洗頭皮
  8. 🚿 第二次清潔（非常重要！）：必須再次使用洗髮精進行第二次頭皮清潔，徹底清除油脂與溶解的污垢
  9. 沖洗乾淨：用大量清水沖洗至完全無油膩感，確保頭皮清爽
  10. 後續護理：可依個人需求使用潤髮產品於髮尾（避開頭皮）

⚠️ 為什麼需要洗兩次？
第一次清洗是將油脂和溶解的污垢帶出來，但此時可能還有殘留。第二次清洗才能徹底清除，確保頭皮清爽不油膩。若只洗一次，可能會感覺頭皮沉重、髮根塌陷。

💡適用情境與建議頻率

  • ✓ 頭皮出油旺盛者：每週1-2次深層養護
  • ✓ 頭皮有不適感或緊繃者：每週1次舒緩調理
  • ✓ 一般頭皮保養：每2週1次預防性養護
  • ✓ 染燙後頭皮照護：染燙後3天可使用，幫助頭皮恢復舒適
  • ✓ 換季時期：季節轉換時加強使用，幫助頭皮適應環境變化

重要注意事項

  • 敏感測試必做：首次使用前，請先取少量塗抹於手腕內側或耳後，觀察15-30分鐘，確認無不適反應再用於頭皮
  • 頭皮有傷口或發炎時禁用：若頭皮有傷口、破皮、毛囊炎、濕疹或任何發炎狀況，請勿使用
  • 本配方含茶樹精油：對茶樹過敏者請勿使用；孕婦使用需謹慎
  • 本配方含柑橘類精油（光敏性）：使用後8-12小時內避免頭皮直接曝曬於陽光下，建議於晚間使用
  • 避免接觸眼睛：使用時請小心避免讓調理油流入眼睛，若不慎接觸請立即以大量清水沖洗
  • 僅用於頭皮，不是髮膜：本產品專為頭皮設計，不適合當作髮膜使用於髮絲上
  • 用量適中即可：不需要大量使用，適量即可達到效果，過量反而難以清洗
  • 靜置時間不宜過長：建議靜置15-20分鐘即可，不需要超過30分鐘
  • 務必清洗乾淨：若清洗不徹底會導致頭皮悶住、髮根扁塌，請確實進行兩次清潔
  • 染燙當天不建議使用：染髮或燙髮當天頭皮較為敏感，建議等待3天後再使用
  • 頭皮敏感者謹慎使用：若平時頭皮容易敏感，建議先從小範圍測試，或稀釋至1.5%濃度使用
  • 兒童使用需調整濃度：12歲以下兒童若需使用，建議稀釋至1%濃度，並縮短靜置時間至10分鐘
  • 孕婦、哺乳期間請諮詢專業：懷孕或哺乳中若想使用，建議先諮詢醫療專業人員
  • 儲存方式：請放置於陰涼乾燥處，避免陽光直射與高溫環境，並遠離兒童可觸及之處
  • 保存期限：開封後建議於6個月內使用完畢（含柑橘類精油，開封後較易氧化）
  • 若有不適請立即停用：使用過程中若出現刺痛、灼熱、紅腫、發癢等不適症狀，請立即用清水沖洗並停止使用

🔬什麼是外泌體？

外泌體（Exosome）是一種奈米級的微小囊泡，能夠幫助肌膚細胞之間的溝通與傳遞營養。簡單來說，就像是肌膚的「營養快遞員」，能將修護成分更有效地送達肌膚深層。

外泌體技術在近年成為高端保養品的新趨勢，因其分子細小、滲透力強，被廣泛運用於肌膚深層修護與活化調理。

臉部使用方法

  1. 清潔肌膚：使用溫和洗面乳徹底清潔臉部，擦乾水分
  2. 開啟安瓶：輕輕折斷安瓶頸部（建議用紙巾包覆避免割傷），將精華液倒於手心
  3. 均勻塗抹：將精華液均勻塗抹於全臉，特別加強在需要修護的部位（如細紋、暗沉區域）
  4. 輕柔按摩：以指腹輕柔按摩2-3分鐘，幫助吸收，按摩方向由內而外、由下往上
  5. 靜待吸收：等待3-5分鐘讓精華完全吸收
  6. 後續保養：可依個人需求接續使用乳液或面霜鎖住精華

💡 建議使用頻率：一般保養每週使用2-3支；加強修護期可每日使用1支（建議連續使用14-30天）

頭皮使用方法

  1. 清潔頭皮：先用洗髮精徹底清潔頭皮，吹至半乾或全乾狀態
  2. 分區塗抹：將頭髮分區，打開安瓶後，用滴管或直接將精華液滴於頭皮上

📖什麼是耳燭？

耳燭（Ear Candle）是一種源自古老文明的傳統芳香照護方式，主要由天然蜂蠟、棉布或亞麻布製成的中空錐形蠟燭。使用時將蠟燭一端輕置於耳道外側，另一端點燃，利用燃燒過程中的溫和熱流與煙道效應，進行芳香照護體驗。

重要提醒：耳燭並非醫療器材，無法清除耳垢或治療耳部疾病。醫學研究指出，耳燭的主要價值在於提供一種放鬆的儀式感體驗，而非實際的生理清潔效果。

🕯️傳統芳療照護用途

以下為傳統芳香照護觀點，非醫療功效宣稱

情緒紓解與放鬆

透過芳香蠟燭燃燒的過程，搭配寧靜環境與輕柔音樂，提供一個專屬的放鬆時光，幫助緊繃情緒獲得舒緩

頭頸部溫暖調理

溫和的熱流可能帶來耳周、頭部區域的溫暖舒適感，就像是做了一個迷你版的溫熱照護

儀式感與正念體驗

耳燭操作過程需要靜躺15-20分鐘，這段時間可以練習深呼吸、冥想，或單純享受當下的寧靜，是一種正念放鬆的體驗

壓力管理的輔助方式

在忙碌生活中，給自己一個「什麼都不做」的時間，單純感受溫暖與寧靜，是壓力管理的一種溫和方式

專業操作流程（僅供受訓學員參考）

⚠️ 警告：以下操作流程僅供受過專業訓練的學員參考，絕對禁止未受訓者自行操作！

  1. 環境準備：選擇安靜、通風良好、無易燃物的安全空間，準備大盆清水、濕毛巾、防護鋁箔盤等安全器材
  2. 個案評估：詳細詢問個案是否有耳部疾病、耳膜穿孔、耳道發炎等禁忌症，確認可進行後才開始
  3. 個案準備：讓個案側躺於按摩床上，使用專用防護鋁箔盤覆蓋於耳部周圍，確保無蠟油滴落風險
  4. 耳燭固定：將耳燭較細的一端輕輕置於耳道外側開口（不進入耳道內！），確保穩定但不用力壓迫
  5. 點燃操作：點燃耳燭較粗的一端，全程不可離開個案身邊，隨時注意火勢與個案反應
  6. 過程監控：耳燭燃燒時間約10-15分鐘，操作者須全程在旁觀察，每3-5分鐘確認個案舒適度
  7. 安全熄滅：當耳燭燃燒至安全標示線（通常距離耳朵10-15公分處）時，立即取下並放入準備好的清水中熄滅
  8. 雙耳操作：一側完成後，協助個案翻身，以相同方式操作另一側
  9. 後續照護：操作完成後，讓個案靜躺5-10分鐘，提供溫水飲用，並叮嚀當日避免碰水、避免用力挖耳
  10. 環境清理：確實清理所有蠟燭殘渣與灰燼，檢查周圍無任何火源殘留

  • 孕婦使用需謹慎：懷孕期間使用刮痧工具需特別小心，某些穴位不宜刺激，建議諮詢專業
  • 臉部刮痧力道要輕：臉部皮膚較為細緻，力道務必輕柔，避免過度拉扯造成皺紋
  • 出痧是正常現象：身體部位刮痧可能會出現紅色或紫色痧點，這是正常的氣血循環反應，通常3-7天會自然消退
  • 但臉部不宜出痧：臉部刮痧力道要輕，不要刮到出痧，以免影響外觀
  • 刮痧後注意保暖：刮痧後皮膚毛孔張開,請注意保暖，避免吹風受涼
  • 刮痧後多喝溫水：有助於代謝循環，建議刮痧後30分鐘內飲用300-500ml溫開水
  • 刮痧間隔時間：同一部位的刮痧建議間隔3-5天，等痧退了再進行下一次
  • 以下情況不宜刮痧：血小板低下、凝血功能異常、正在服用抗凝血藥物、皮膚過敏、嚴重心血管疾病、飯後1小時內
  • 天然石材需小心保護：粉晶、白玉、砭石都是天然石材，避免摔落、碰撞，以免破損
  • 溫差影響：天然石材遇到劇烈溫差可能會產生裂紋，不要從冰箱拿出後立即用熱水沖洗
  • 個人衛生：若用於個人居家使用，使用前後請清洗雙手
  • 專業服務建議：若用於提供個案服務，建議投保相關責任險
  • 非醫療用途：本工具組為美容照護輔助工具，不具任何醫療用途，無法治療疾病
  • 若有不適請立即停止：使用過程中若個案感到疼痛、頭暈、噁心等不適，請立即停止操作
